# SQL-Python-Ecommerce-Project

Project Overview - 
This is a SQL + Python e-commerce analysis project. I did this project in Jupyter Notebook for learning and practice. SQL was used to fetch data from the database, and Python was used to analyze and visualize the data.

Database Connection -
The e-commerce data is stored in a MySQL database. I connected MySQL with Python using mysql.connector and executed SQL queries inside Jupyter Notebook.

Tools Used - 
MySQL
Python (Pandas, Matplotlib, Seaborn)
Jupyter Notebook

What I Did - 
Connected MySQL database with Python
Wrote SQL queries to fetch data
Loaded query results into Pandas DataFrames
Analyzed data using Pandas
Created basic charts for insights

Analysis Includes -
Orders and sales analysis
Category-wise revenue
Monthly and yearly trends
Customer and seller analysis

Learning Outcome - 
This project helped me improve my SQL querying, Python data analysis, and data visualization skills.
